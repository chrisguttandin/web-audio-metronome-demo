This is a demo of a metronome which uses the Web Audio API to achieve a very high precision. It's basically following the technique described by Paul Adenot in his blogpost named "A robust metronome using the Web Audio API". The metronome is connected to a timing-object which is itself connected to a timing-provider. That means multiple instances of this page should be in sync with each other.
